# 💧 Sistema Hidráulico

Site de uma empresa de produtos hidráulicos com área de login e consulta de produtos.

---

## 🔐 Login

Acesse a página `login.html` e utilize as credenciais:

> **E-mail:** `admin@admin`  
> **Senha:** `admin`

---

## 🔍 Consultar Produto

Após o login, clique em **Consultar Produto** no menu lateral, digite o nome do produto e clique em **Buscar**.

As informações exibidas são: imagem, nome, preço e descrição.

---

## 🚀 Como rodar

Clone o repositório e abra o arquivo `index.html` no navegador.

```bash
git clone https://github.com/SamuelLacerdx/page_hidraulica.git
```
